Como usar o Sistema de Relatórios

1. Filtros: Configure o período, forma de pagamento e tipo de plano desejados.

2. Configurações: Escolha o tipo de relatório e formato de exibição.

3. Gerar: Clique em "Gerar Relatório" para visualizar os dados.

4. Exportar: Use os botões para exportar em PDF, imprimir ou compartilhar.
